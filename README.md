# Ai-CCTV-System

인공지능을 이용한 CCTV 시스템의 소개이다.

# 🛠 개발 도구
* **Jupyter Notebook**
# 📚 라이브러리 및 버전
* **Keras**: 2.13.0
* **TensorFlow**: 2.13.0
* **OpenCV**: 4.8.0
* **NumPy**: 1.24.3
* **h5py**: 3.9.0

# 💻 사용 언어
* **Python**

# 기능 및 특징
* 초음파 센서를 이용하여 거리가 15cm이상이면 직진을 15cm미만이면 좌회전을 하도록 모터 제어를 설정하였다.
